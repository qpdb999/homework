# 나의 첫 웹페이지

## 프로젝트 소개
HTML과 CSS를 사용해 나를 소개하는 웹페이지를 만들었습니다.

## 사용한 파일
- index.html : 웹페이지의 구조
- style.css : 글자 색과 배치 등 디자인 설정

## 실행 방법
GitHub Pages를 이용해 웹사이트로 공개하였습니다.
주소: https://아이디.github.io/저장소이름/

## 수정한 내용
- 배경색을 파란색으로 변경함
- 제목 글자를 크게 수정함
- 나를 소개하는 문단을 추가함

## 느낀 점
어렵지만 직접 코드를 바꾸면 웹페이지가 변하는 것이 재미있었습니다.
